# Redesign Devalor Cobranças — Relatório de Melhorias

**Preparado para:** Devalor Cobranças
**Data:** Abril de 2026
**Versão:** 1.0

---

## Sumário Executivo

Este documento apresenta as melhorias realizadas no redesign da landing page da Devalor Cobranças, comparando o site anterior com a nova versão. O objetivo do projeto foi criar uma presença digital que transmita **confiança, profissionalismo e clareza** — valores essenciais para uma empresa que lida com o patrimônio financeiro de seus clientes.

As mudanças abrangem três pilares: **mensagem (copywriting)**, **design visual** e **uso estratégico de animações**.

---

## 1. Melhorias na Mensagem

### 1.1 De funcionalidades para resultados

| Aspecto              | Site Anterior                       | Novo Site                                          |
| -------------------- | ----------------------------------- | -------------------------------------------------- |
| **Título principal** | "Custo zero para sua empresa!"      | "Recupere até 80% dos seus créditos inadimplentes" |
| **Abordagem**        | Foco em características do serviço  | Foco no resultado que o cliente obtém              |
| **Subtítulo**        | Mistura de estatísticas e buzzwords | Proposta de valor direta e sem ruído               |

**Por que isso importa:** O visitante que chega ao site quer saber primeiro _o que ele ganha_, não _como o serviço funciona_. O título anterior ("Custo zero") é uma característica — importante, mas secundária. O novo título comunica o benefício principal de forma imediata: recuperação de créditos. O "custo zero" agora aparece como reforço no subtítulo, no lugar certo da hierarquia de informação.

### 1.2 Eliminação de repetição

O site anterior repetia as mesmas informações múltiplas vezes:

- **"Custo zero"** aparecia 4 vezes em seções diferentes
- **"Resultados de 40% a 80%"** aparecia 3 vezes
- **"Sem mensalidade ou taxa de contrato"** aparecia 3 vezes

Essa repetição fazia a página parecer inflada e pouco profissional. No novo site, **cada benefício é declarado uma única vez, na seção apropriada**, com contexto suficiente para ser convincente. O resultado é uma página mais curta, mais objetiva e mais persuasiva.

### 1.3 Nova seção "Como Funciona"

O site anterior não explicava o processo de cobrança em nenhum lugar — o visitante precisava ler o FAQ para entender como contratar. Isso é uma barreira significativa para conversão.

O novo site inclui uma seção dedicada com 4 passos claros:

1. Cadastre seus inadimplentes no portal
2. A equipe inicia as 5 fases de cobrança
3. Acompanhe tudo em tempo real pelo portal
4. Receba 100% do valor capital recuperado

**Impacto:** Reduz a incerteza do visitante e facilita a decisão de contato.

### 1.4 CTAs mais estratégicos

| Site Anterior                        | Novo Site                                          |
| ------------------------------------ | -------------------------------------------------- |
| "QUERO CONTRATAR" (compromisso alto) | "Solicite uma Proposta Grátis" (compromisso baixo) |
| "Entrar em contato" (genérico)       | "Fale com a Equipe Comercial" (específico)         |
| CTA único repetido                   | CTAs variados por contexto de seção                |

A mudança de "QUERO CONTRATAR" para "Solicite uma Proposta Grátis" reduz a barreira psicológica do primeiro contato. O visitante não precisa se comprometer com uma contratação — apenas solicitar informações.

## 2. Melhorias no Design

### 2.1 Identidade visual consistente

| Aspecto             | Site Anterior                             | Novo Site                                                                                           |
| ------------------- | ----------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Aparência geral** | Template WordPress genérico com Elementor | Design fintech moderno e personalizado                                                              |
| **Paleta de cores** | Múltiplas cores sem hierarquia clara      | Azul marinho (#1A3960) como cor primária, verde (#2ECC71) como destaque, preto (#0D0D0D) para texto |
| **Tipografia**      | Fontes padrão do template                 | Montserrat — moderna, legível, profissional                                                         |
| **Espaçamento**     | Seções comprimidas, pouco respiro         | Espaçamento generoso entre seções e elementos                                                       |

### 2.2 Navegação

**Antes:** Menu genérico sem âncoras para seções da página. No rodapé, múltiplos números de WhatsApp para departamentos diferentes, gerando confusão.

**Depois:** Navegação fixa no topo com fundo escuro (contraste adequado para o logo), links âncora para cada seção, e um único CTA de contato visível. Menu mobile com animação suave de abertura/fechamento. Botão flutuante de WhatsApp sempre acessível.

### 2.3 Prova social fortalecida

| Elemento              | Site Anterior                                  | Novo Site                                                   |
| --------------------- | ---------------------------------------------- | ----------------------------------------------------------- |
| **Depoimentos**       | Texto puro com nome da empresa                 | Cards estruturados com avatar, empresa e cargo              |
| **NPS**               | Mencionado mas sem número visível              | Score NPS 10 em destaque com card visual dedicado           |
| **Métricas**          | Contadores que exibiam "0" quando o JS falhava | Números animados com valor estático como fallback           |
| **Logos de clientes** | Carrossel com apenas 1 logo visível            | Grade estática com múltiplos nomes visíveis simultaneamente |

### 2.4 Diagramas e métricas visuais

O novo site inclui duas seções de dados visuais que não existiam anteriormente:

- **"Números que falam por si"** — Barras de progresso animadas mostrando taxa de recuperação por faixa de vencimento, e gráficos radiais com indicadores de performance (recuperação, repasse, satisfação, localização de devedores)
- **"As 5 Fases da Cobrança Devalor"** — Timeline visual animada detalhando a metodologia progressiva, com canais utilizados em cada fase

**Impacto:** Transforma dados abstratos em evidência visual concreta. O visitante _vê_ os resultados em vez de apenas ler sobre eles.

### 2.5 Seção de diretores

A apresentação dos diretores Rodrigo e Caio foi modernizada com fotos em cards limpos, efeito hover sutil, e hierarquia clara de nome e cargo. Humanizar a liderança aumenta a confiança — o visitante sabe quem está por trás da empresa.

### 2.6 Responsividade

O site anterior dependia de widgets Elementor que nem sempre se adaptavam bem a telas menores. O novo site foi construído com abordagem mobile-first, garantindo que cada seção funcione corretamente em celulares, tablets e desktops.

## 3. Uso Estratégico de Animações

### 3.1 Por que menos é mais em fintech

Empresas do setor financeiro que reduzem o uso de animações decorativas fazem isso por razões fundamentadas:

**Confiança e seriedade.** A Devalor lida com o patrimônio financeiro dos seus clientes. Animações excessivas podem transmitir uma impressão de informalidade ou falta de solidez. Um site que apresenta informações de forma limpa e direta comunica estabilidade — exatamente o que uma empresa de cobrança precisa transmitir.

**Foco no conteúdo.** Quando um potencial cliente está avaliando se deve confiar seus créditos inadimplentes à Devalor, cada animação compete pela atenção dele. Reduzir movimentos decorativos mantém o foco na mensagem e na decisão.

**Velocidade percebida.** Animações que atrasam a exibição do conteúdo (mesmo por frações de segundo) fazem o site parecer mais lento. Estudos de conversão mostram consistentemente que velocidade percebida se correlaciona com confiança e taxa de cadastro.

**Acessibilidade.** Uma parcela significativa de usuários possui sensibilidade a movimento (distúrbios vestibulares). Sites com animações excessivas podem causar desconforto físico. Ser conservador com movimento garante que nenhum visitante seja excluído.

**Sinal de maturidade.** Empresas em estágio inicial tendem a usar muitas animações para parecer "modernas". Empresas estabelecidas simplificam. Animações sutis e funcionais comunicam sofisticação mais do que efeitos visuais elaborados.

### 3.2 O que mantivemos e por quê

As animações no novo site seguem o princípio **"movimento para função, não para decoração"**:

| Animação                           | Justificativa                                                               |
| ---------------------------------- | --------------------------------------------------------------------------- |
| **Contadores numéricos** (hero)    | O número subindo de 0 a 600+ _é_ o conteúdo — reforça a métrica visualmente |
| **Barras de progresso** (métricas) | A barra preenchendo comunica proporção de forma intuitiva                   |
| **Gráficos radiais** (métricas)    | O anel desenhando transmite progresso e completude                          |
| **Timeline** (processo)            | A sequência animada reforça a progressão das 5 fases                        |
| **Padrão de pontos** (hero)        | Movimento diagonal sutil cria profundidade sem distrair                     |
| **Botão WhatsApp**                 | Entrada com spring chama atenção para o canal de contato principal          |

### 3.3 O que removemos ou simplificamos

- Sem efeitos de parallax pesados
- Sem carrosséis automáticos (escondem conteúdo)
- Sem animações que bloqueiam a leitura do conteúdo
- Transições de hover sutis em vez de transformações dramáticas

## 5. Resumo Comparativo

| Critério              | Site Anterior                      | Novo Site                                    |
| --------------------- | ---------------------------------- | -------------------------------------------- |
| Mensagem principal    | Funcionalidade ("custo zero")      | Resultado ("recupere até 80%")               |
| Repetição de conteúdo | Alta (mesmos pontos 3-4x)          | Eliminada                                    |
| Seção "Como Funciona" | Inexistente                        | 4 passos claros                              |
| Prova social          | Fraca (texto puro, NPS sem número) | Forte (cards, NPS visível, métricas visuais) |
| Dados visuais         | Contadores que mostravam "0"       | Barras, gráficos radiais e timeline animados |
| Design                | Template genérico                  | Fintech moderno com identidade própria       |
| Animações             | Nenhuma significativa              | Funcionais e estratégicas                    |
| Responsividade        | Parcial (Elementor)                | Completa (mobile-first)                      |
| Velocidade percebida  | Lenta (widgets pesados)            | Rápida (Next.js otimizado)                   |
| CTAs                  | Genéricos e de alto compromisso    | Específicos e de baixa barreira              |
